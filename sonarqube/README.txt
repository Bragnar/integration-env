


SonarQube C and C++ plugins are not available on the MArketPlace, they must be installed manually.
Installation prodecure is available on their GitHub : https://github.com/SonarOpenCommunity/sonar-cxx


The current version (1.2.2) does not support SonarQube v7.7, see issue #1695 (https://github.com/SonarOpenCommunity/sonar-cxx/pull/1695)

Artifacts for the plugin including the sopport for SonarQube 7.7 are available here :
https://ci.appveyor.com/project/SonarOpenCommunity/sonar-cxx/builds/23281379/artifacts
